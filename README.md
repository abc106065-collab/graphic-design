## 🧭 How to use it

### 1. Start a new design
Open graphic-design and choose the type of design you want. You can pick a banner, flyer, logo, poster, or packaging layout.

### 2. Enter your text
Type a short prompt that explains what you want. For example:
- clean logo for a coffee shop
- modern poster for a music event
- product box design for skincare
- summer sale banner for a clothing store

### 3. Choose a style
Pick a style that fits your project. Common styles include:
- modern
- minimal
- bold
- clean
- colorful
- corporate
- creative

### 4. Generate the design
Click the generate button. The app builds a design based on your text and style choice.

### 5. Review and edit
Look over the result. If needed, change the text, colors, or layout and create a new version.

### 6. Save your file
Save the final design to your PC. You can use it for online posts, print jobs, or client work.

## 🛠️ Basic help

### The app does not open
Check that the download finished fully. Try opening the file again. If it is inside a ZIP file, unpack it first.

### The design looks wrong
Change your prompt and try again. Use fewer words and make the main subject clear.

### The image quality looks low
Use a prompt with a clear style and fewer extra details. Try a new render with a higher-quality option if the app offers one.

### The text is not placed well
Shorten the text and use a different layout style. Large blocks of text can be hard to fit in a design.
